    🎯 PASSO A PASSO COM IMAGENS (ONDE CLICAR)
═══════════════════════════════════════════════════════════════

PASSO 1: ABRIR DEPLOYMENTS
───────────────────────────────────────────────────────────────
Na barra lateral ESQUERDA do Replit:
→ Procure o ícone que parece um FOGUETE (🚀)
→ OU procure a palavra "Deployments"
→ CLIQUE nele

PASSO 2: LOCALIZAR AS OPÇÕES
───────────────────────────────────────────────────────────────
Você vai ver uma tela com seu deployment atual.

Procure por:
→ Botão "Republish" (grande, azul)
→ AO LADO dele, tem "..." (três pontinhos)
→ OU "More" / "Options"

PASSO 3: FORCE REBUILD
───────────────────────────────────────────────────────────────
CLIQUE nos três pontinhos (...)

No menu que abre, procure:
→ "Force rebuild"
→ OU "Rebuild from scratch"
→ OU "Clear cache and rebuild"

CLIQUE nessa opção

PASSO 4: CONFIRMAR
───────────────────────────────────────────────────────────────
Pode aparecer uma mensagem perguntando se tem certeza.

→ CLIQUE "Yes" / "Confirm" / "Rebuild"

PASSO 5: AGUARDAR
───────────────────────────────────────────────────────────────
Você vai ver:
→ Status: "Building..." ou "In progress"
→ Pode demorar 3-5 minutos
→ Aguarde até aparecer: "Active" ou "Deployed"

PASSO 6: TESTAR
───────────────────────────────────────────────────────────────
Após status "Active":

1. Abra navegador ANÔNIMO (Ctrl+Shift+N)
2. Acesse: https://mixapp.digital/phone-auth
3. Preencha: email, telefone, senha
4. Clique "Criar Conta"

✅ DEVE FUNCIONAR!

═══════════════════════════════════════════════════════════════

❓ SE NÃO ENCONTRAR "FORCE REBUILD":
───────────────────────────────────────────────────────────────

ALTERNATIVA 1:
→ Clique "Create new deployment"
→ Aguarde build finalizar
→ Configure mixapp.digital para novo URL

ALTERNATIVA 2:
→ Vá na aba "Shell"
→ Digite: touch .force-rebuild
→ Pressione ENTER
→ Volte em Deployments → Republish
